# React + Vite

Proyecto que utiliza React con Vite como entorno de desarrollo.

---

# Enrutamiento

Para habilitar el uso de rutas en React (como las del navbar), ejecutar en la terminal:

```bash
npm install react-router-dom@latest

```
---

# BACKEND 

## Ingresar a la carpeta backend y escribir los siguientes comandos:

```bash
npm init -y
npm install express mssql cors
npm install bcrypt
npm install multer
```
Nota:
- **express** es un framework para crear el servidor backend de forma sencilla y eficiente.  
- **mssql** permite conectar y realizar consultas a bases de datos SQL Server desde Node.js.
- **cors** permite conectar el proyecto con SQL Server evitando problemas de CORS.
- **bcrypt** permite encriptar contraseñas.
- **multer**  permite guardar archivos tipo imágenes evitando errores al interpretarlos como binarios.

# FRONT-END

## Instalación de FontAwesome
```bash
npm install @fortawesome/react-fontawesome @fortawesome/free-solid-svg-icons
```
Nota: No olvidar **poner** el enlace de fontawesome en index.html
